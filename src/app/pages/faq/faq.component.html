Watching is legal. We also do not host the anime, we link to them. AnimetTV is more of a google search engine for anime in the web.

 External Player / Sandbox Detected (Player not working). Only for   Chrome

Troubleshoot player error. Only happens in chromium browsers.

Edite chrome setting -> Download PDFs or Use Firefox or a mobile device.
chrome://settings/content/pdfDocuments
It's because the external player using pdf detection to check if the ads are being block. If the pdf feature disabled, the check will pass, and not block the stream. If problem still presists please use Firefox, any mobile device or DM Admin in our Discord Server and he/she will guide you fixing the issue. Email
